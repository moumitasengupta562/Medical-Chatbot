# Medical-Chatbot

# how to run?
### STEPS 01- Create a conda environment after opening the repository

Clone the Repository

```bash
git clone: https://github.com/moumitasengupta562/Medical-Chatbot.git
```
### STEP 01- Create a conda environment after opening the repository

```bash
conda create -n SM Nexus python=3.10 -y
```

```bash
conda activate SM Nexus 
```


### STEPS 02- install the requirement
```bash
pip install -r requirements.txt
```
